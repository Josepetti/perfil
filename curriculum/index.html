Projetos Realizados no Curso

Projeto 1: Tela de Login

A tela de login é a interface que permite a autenticação de um usuário em um sistema, site ou aplicativo. O seu desenvolvimento envolve a criação do design visual (UI/UX) e a programação de ponta a ponta (frontend e backend) para validar credenciais, garantir a segurança e gerenciar sessões.

Projeto 2: CRUD em Python

CRUD é o acrônimo para as quatro operações básicas em um sistema de banco de dados: Create (Criar), Read (Ler), Update (Atualizar) e Delete (Excluir). Em Python, isso é feito facilmente usando o banco de dados nativo SQLite.

Projeto 3: Contador

Para criar um contador em Java, você pode usar uma variável inteira (int) com laços de repetição como for ou while, ou instanciar uma classe de forma Orientada a Objetos.

Projeto 4: To-Do List

A To-Do List é uma aplicação que permite aos usuários criar, visualizar, editar e excluir tarefas. Ela pode ser desenvolvida usando HTML, CSS e JavaScript para o frontend, e uma linguagem de backend como Python ou Java para gerenciar os dados.

Projeto 5: Sistema com Bootstrap

O desenvolvimento de um sistema utilizando o Bootstrap envolve a criação de uma interface responsiva e moderna, aproveitando os componentes pré-construídos e as classes utilitárias do framework para garantir uma experiência de usuário consistente em diferentes dispositivos.

Projeto 6: Cronometro

O cronômetro é uma aplicação que permite medir o tempo decorrido entre um evento de início e um evento de parada. Ele pode ser implementado usando JavaScript para criar a funcionalidade de contagem regressiva ou progressiva, e pode incluir recursos adicionais como pausa, reinício e exibição do tempo em formato legível.

Projeto 7: Contador e Alternancia de Tema

Este projeto combina a funcionalidade de um contador, que pode ser incrementado ou decrementado, com a capacidade de alternar entre temas claro e escuro. A alternância de tema é geralmente implementada usando JavaScript para adicionar ou remover classes CSS que definem as cores e estilos do tema.

Projeto 8: Calculadora Simples

A calculadora simples é uma aplicação que permite aos usuários realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão. Ela pode ser desenvolvida usando HTML para a estrutura, CSS para o estilo e JavaScript para a lógica de cálculo e interação do usuário.

Projeto 9: Projeto HTML e CSS3

Este projeto consiste em criar uma página web utilizando HTML e CSS3, focando em estruturação e estilização de elementos.

Projeto 10: Projeto ultilizando o Bootstrap

Este projeto envolve a criação de uma página web utilizando o framework Bootstrap, aproveitando seus componentes e classes para garantir uma interface responsiva e moderna.

Projeto 11: Manipulando colunas com Bootstrap

Este projeto demonstra como manipular colunas usando o sistema de grid do Bootstrap, permitindo criar layouts flexíveis e responsivos para diferentes tamanhos de tela.
